Structure of the repository

* `cosmo_models/`: never used
* `GW/`: main code for gravitational wave analysis
* `initial_notebooks/`: initial exploration and analysis notebooks
* `tabularassa.py`: Computing contours for LGRB
* `2tabularassa.py`: Creating animation to see the effect of dof/no-dof and the extra scatter
* `joint-constraint.ipynb`: notebook for joint constraints combining GRB and GW
* `X_chi_surface.npy` and `X_mask.npy`: data files for chi surface and mask
* `README.md`: this file
